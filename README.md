# Fun Shell Scripts!

## How to USE?

### step: 1
```sh
git clone <this-repo>
```
### step: 2
```sh
cd <this-repo-name>
```

### step: 3
```sh
bash <script-name>
```

### --- or ---

```sh
chmod +x <script-name>
```

```sh
./<script-name>
```
 
## game.sh & functions.sh:
  * game.sh - Main file
  * functions.sh - Contains all the functions needed for the game.sh
  * Inspiration: [NetworkChuck](https://www.youtube.com/watch?v=Fq6gqi9Ubog)
